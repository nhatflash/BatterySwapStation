# EV Battery Swap Station

**Business Flow**
----------------------------------------------------------
*User scheduled a battery swap transaction*
1. User schedules a date and time on a station for swapping
2. The station's staff confirms the scheduled swap transaction request.
3. User is allowed to pay the transaction via cash or VNPay
4. The station's staff confirm the arrival of the user when they arrive at the station
5. The station's staff start the battery swapping process, an idle swap bay will be taken
6. The station's staff end the battery swapping process and release the swap bay for another swap turn
7. User submits the feedback after swapping ended.


*User arrives at the station directly*
1. The station's staff manually create a walk-in swap transaction for the user
2. The station's staff confirms the scheduled swap transaction request.
3. User is allowed to pay the transaction via cash or VNPay
4. The station's staff confirm the arrival of the user when they arrive at the station
5. The station's staff start the battery swapping process, an idle swap bay will be taken
6. The station's staff end the battery swapping process and release the swap bay for another swap turn
7. User submits the feedback after swapping ended.
